Frequently Asked Questions (FAQ'S)

1. What is HTML?
HTML (HyperText Markup Language) is the foundation of all websites. It defines the structure of web content.
2. What is the purpose of CSS?
CSS (Cascading Style Sheets) enhances the appearance of HTML by controlling layout, colors, and fonts.
3. What is JavaScript used for?
JavaScript adds interactivity to websites—like clickable menus, form validation, animations, and more.
4. What is a responsive website?
A responsive website adjusts its layout and content to fit different screen sizes—like phones, tablets, and desktops.
5. Is web development a good career?
Yes! Web development offers creative freedom, strong demand, and many paths for growth in tech fields.
